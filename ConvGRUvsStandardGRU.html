Spatial Awareness: Standard GRU vs ConvGRU

Visualizing how Neural Networks ingest a 2D Video Frame.

Frame Size: 4x4 Pixels

Standard GRU (Loss of Space)

The 2D image must be flattened into a 1D array to be multiplied by a dense matrix. Neighboring pixels are torn apart.

Dense Weight Matrix (W) Requires 16x16 = 256 parameters
Watch how the blue "subject" pixels (indexes 5, 6, 9, 10) are ripped apart. The network no longer knows they were next to each other!

ConvGRU (Spatial Preservation)

The image stays in a 2D grid. A Convolutional Kernel (sliding window) processes the data, inherently understanding what pixels are neighbors.

3x3 Kernel (W) 9 params
Shared
The 3x3 kernel perfectly covers the 2D neighborhood. It processes the "head" and "body" pixels simultaneously, retaining the shape geometry.
